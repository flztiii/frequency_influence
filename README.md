# Requirements

- Python 3.5.x

- numpy

- scipy

- matplotlib

- msttcorefonts

# Usage

1. Clone this repo

> git clone https://github.com/flztiii/frequency_influence.git

2. install fonts

> sudo apt-get install msttcorefonts

3. run the program

> python3 frenet_optimal_trajectory.py
